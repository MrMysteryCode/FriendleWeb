Friendle Privacy Policy

Effective date: January 29, 2026

Overview

Friendle is a daily guessing game inspired by Discord server activity. We are committed to privacy-first design. The game only displays anonymized, aggregated insights from the previous day's activity.

Data We Use

  • Daily aggregates such as message count ranges and activity windows.
  • De-identified snippets such as scrambled quotes and pixelated media.
  • Public Discord metadata like account age ranges.

Data We Do Not Use

  • Private messages (DMs) or content from opt-out channels.
  • Roles, sensitive attributes, or personal profile details.
  • Exact message timestamps or raw message history.

How Data Is Processed

Data is collected by a Discord bot and transformed into daily puzzle clues. The bot caches the daily result to avoid re-scanning messages and to ensure each day has a single shared puzzle.

Retention

Daily game data is stored only as long as needed to serve the current day's puzzle. Historical data is not retained in a way that allows reconstruction of past messages.

Sharing

We do not sell or share user data with third parties. The only output shown to players is anonymized and aggregated puzzle data.

Contact

If you have questions about this policy or wish to opt out of data usage in your server, contact your server admin or the Friendle team.
